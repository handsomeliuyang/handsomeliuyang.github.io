    搭建过程

    1. Github配置

      1. 在Github上申请一个帐号
      2. 创建一个repository，其命名规则有两种
        1. <你的用户名>.github.io // 那你的博客地址就是 http://<你的用户名>.github.io 推荐
        2. <任意名称> // 那你的博客地址将是：http://<你的用户名>.github.io/<任意名称>
      3. 使用ssh连接，配置ssh的公钥和私钥，以后连接github不用再输入密码

    2. Hexo安装

      1. 请按最新官网安装并配置Hexo，具体请看：https://hexo.io/
